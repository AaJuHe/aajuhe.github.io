Semantic Versioning

By Aaron Hearne

Select Your Editor Web Hosting Chrome DevTools Front-End Frameworks Semantic Versioning

I always noticed the version numbers of certain programs and suites as well as the updates made to the program in the current version. However, prior to watching the video, I was not too sure about what the numbers meant. After watching the video, I now have a better idea as to what the numbers mean and how it affects the programs and the capabilities of said programs. In addition, it lets me know if the current program is outdated and that an upgrade is the best course of action.

The program I chose is LibreOffice, which is an open source suite of programs very similar to Microsoft Office. It is most common in Linux; however, it can be used for Windows as well. The most current version is 5.3.0, which is a change from 5.2.5. According to the video about semantic versioning, while there have been some new features added to the suite, there has not been anything that would break the API of the suite; therefore, there has not been any major changes.
