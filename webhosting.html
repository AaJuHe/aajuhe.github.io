Web Hosting

By Aaron Hearne

Select Your Editor Web Hosting Chrome DevTools Front-End Frameworks Semantic Versioning

The sites I came across may not have the best features, may not be the cheapest, nor may I use them should I decide to have my own site. Having said that, I picked them for two reasons: there were a few things about the sites I found intriguing, and I wanted to make sure I did not pick sites others have already used.

The free web hosting site I chose to go with is Weebly. It has some features that I found cool such as drag and drop as well as the ability to edit content on a site via a mobile device. The latter could come in handy if there is something that needs to be added in a pinch and one is not near a PC or laptop. In addition, it offers the ability to create different sites for eCommerce and business; however, the ability to create these sites they have to be paid for.

As for the paid hosting site, I went with Hostwinds. It has three types of sites: basic, advanced, and ultimate. It appears that all three have the same features – such as unlimited bandwidth and space, unlimited email accounts, etc – with the main difference being the number of add-on domains. The basic package allows zero, advanced allows three, and ultimate has unlimited add-on domains. As far as the pricing is concerned, the basic package is about $54/year, the advanced package is $66/year, and the ultimate package is $78/year. If add-on domains is not an issue, the best deal would be the basic package.
