Chrome Dev Tools

By Aaron Hearne

Select Your Editor Web Hosting Chrome DevTools Front-End Frameworks Semantic Versioning

After watching a number of Chrome Developer Tools on YouTube, the video that caught my attention the most was one by Amit Agarwal. Although his video was shorter than most of the others, I was able to learn about some of the techniques I would use the most in his video. One that stood out to me was the ability to edit the text directly on the webpage using a command in the Console section. This makes it far more convenient than having to search through possibly many HTML code, especially if the edit is a quick one. Another cool thing about the Console was the ability to do other tasks such as calculations and JavaScript commands.

In addition to that, I thought it was pretty cool to be able to see what a webpage would look like if printed as well as changing the layout of a webpage to different operating systems such as versions of iOS and Android. Overall, I would say this was a good experience.
