Project 0

By Aaron Hearne

Introduction

Hello. I would like to welcome everyone to my Project 0 website. As hinted in the title, this is a project for my Web Technologies class at Fresno City College. The purpose of this project is to show how much I have learned in this class and how well I applied said knowledge to this project. I must say this was more difficult than I had anticipated; however, I believe I applied a good portion of what I learned while working on this website. Fortunately, this site did not have to be built from scratch, for there was Bootstrap, arguably the most popular front-end framework that did most of the work code-wise. In addition, there were templates created in Bootstrap that took care of the formatting, so the only thing that needed to be added was the content.

As for the site, itself, most the content is based on the lessons and assignments that were done during the course. Linked below is the site map plan which I tried to base the site on. I tried to organize the content in certain categories the best I could. Hopefully, navigation would not be too much of a problem for everyone. With that said, I hope you enjoy your visit to my Project 0 and that you would walk away with some impressions and thoughts, whether good or bad, about Project 0.
