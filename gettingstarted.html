Getting Started with GitHub

By Aaron Hearne

Getting Started Git for your OS Clone GitHub Repo Push GitHub Repo Push Template Directory

Admittedly I struggled for a bit while trying to set up the GitHub page. The main reason was because I did not realize that the name of the site had to match the GitHub username. Because I had created a GitHub account for CIT-15 last semester, I did not want to use the same directory for fear of overriding my previous project. After trying to have my site display unsuccessfully a few times, I searched for help and that is when I learned that the site and username had to match. In order to have a new site and override the old project, there was no way I could use index.html twice. Instead, I used webtech.html for the project of this class. Needless to say, this worked, as evident by the link below.

GitHub Page

It is too bad that different repository names cannot be used under one username. Having to create different extensions is a bit of an annoyance; however, despite that GitHub is a very useful tool for anyone interested in web developing or hosting.
