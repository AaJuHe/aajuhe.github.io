Become a Web Developer

By Aaron Hearne

Become a Web Developer Full Stack vs. Front End Web Development Jobs UI/UX Designer Web Technology Fundamentals

Since I took CIT15 last semester and tried to do some web developing prior to CIT15, I was pretty familiar with the basic front-end terms such as HTML, CSS, JavaScript, and jQuery. In addition, I was pretty familiar with the other scripted languages such as Python, Ruby (although I have never used them before), etc. Finally, I also knew about the relational databases since I have taken a few database classes.

On the flip side, there were some terms I have heard before but never really understood them. One of them was JSON. I knew it had something to do with JavaScript, but I did not know to what extent. It turns out it is a data-interchange language that is beneficial for both humans and machines. Since JSON is text-based, it is independent and can be used with just about any programming language out there. If I become more interest in back-end web development, this is something I may want to use. Below is what I believe is the official website:

JSON

Something else that caught my attention was the software used to create interactive UIs. The person in the video strongly recommended React, so I looked it up and saw how useful it really was. I may look into the others, but from what I have seen so far, it may be difficult to go with anything other than React.

React

The main thing I took from the video is how many options there are and how much one needs to learn in order to be a top notch web developer, especially those going into the back-end or dev ops careers.
