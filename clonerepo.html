At first it was a struggle trying to add the commit change because for some reason Git did not recognize the username or email address of my account. Once those two little nuisances were taken care of, I was able to engender a successful Git log.

As for myself, this is definitely something I want to learn more about because it will add a new weapon for my repertoire. With Project 0 coming up, I want to be able to learn as much as possible with Git Bash.
