Web Development Tools

By Aaron Hearne

Like most, if not all professions, web developers need to have the right tools to be successful at their crafts. By far the most important tool is a text editor. Without a good editor, it would be difficult to write the content and see the layout of the site. Another tool that could come in handy is a developer tool to see how the content is formatted. Seeing that Google Chrome is the most popular web browser, any web developer would benefit from Chrome Dev Tools. Before one can use Chrome Dev Tools, there needs to be a place to host the website or application. Finding a solid web host is crucial to web development. Something else that comes in handy is a framework for those who do not have the time to build a site completely from scratch. Finally, although not a tool, semantic versioning is good to know if one wants to know what the latest version of what something is. Enjoy seeing the possible tools to use after browsing this section.
